# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please send an e-mail to **contact@baderidris.com**.

All security vulnerabilities will be promptly addressed.

## Supported Versions

Currently, only the latest version of this portfolio is supported for security updates.

## Preferred Languages

We can communicate in:
- English
- Arabic (العربية)
- Spanish (Español)
